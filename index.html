Copy the host ID from the GimKit session



Instructions

1. Go to your active GimKit session as a host.

2. Copy the URL that looks like this: https://www.gimkit.com/host?id=YOUR_CODE.

3. Paste it into the input field above and hit "Submit Host URL."

4. You will be redirected to the answers page for real-time results.
